Marvel Rivals website

A Marvel Rivals-themed fan website built with Angular 21. It features a bold black-and-gold aesthetic styled after the game's visual identity, with three main pages: Home, Characters, About The site uses angular (skewed) UI shapes, gold accents, and custom fonts (Racing Sans One, TASA Orbiter) to match the game's action-combat look and feel..

Key Features: Angular 21 SPA with 4 pages (Home, Characters, Character Detail, About), responsive design, custom CSS animations, and a character database fetched from a mock JSON API.
